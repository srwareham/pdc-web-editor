What is Pebble PDC?

Pebble Draw Commands (PDC) are optimized vector instructions used on Pebble watch displays (e.g. Pebble Time). They are lightweight alternatives to animated APNG files, supporting scalable points, colors, strokes, and animations natively on hardware.

Binary Structure:

  • Magic Code: PDCI for single images; PDCS for sequence frames.
  • Color Palette: Custom 64-color 8-bit argb color format.
  • Draw Command: Supports Paths, Circles, and Precise 13.3-fixed-point Paths.

How to Use the Web Tool:

  • Import files: Drag & drop or upload existing .pdc resources or .svg files. Bezier curve nodes will flatten natively!
  • Interactive Canvas: Select elements in the inspector and **drag points directly on the screen** to position them.
  • Watch Buttons: Tap the interactive physical buttons on the watch bezel to emulate player commands directly (Up/Down for frames, Select for play/pause, Back to reset).
PRO TIP: Precise Path (Type 3) allows sub-pixel coordinates for smooth transitions. Points are automatically divided/multiplied by 8 inside the binary coder!
